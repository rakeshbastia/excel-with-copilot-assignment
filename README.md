# 📊 Excel With Copilot Assignment

## 📌 Overview
This assignment demonstrates the use of Microsoft Excel 
integrated with Copilot AI to perform data analysis tasks 
on a sales dataset of 350 orders (Jan–Dec 2023).

## 🛠️ Tools Used
- Microsoft Excel
- Microsoft Copilot (AI Assistant)
- OneDrive

## 📋 Assignment Tasks
| Q No. | Task |
|-------|------|
| Q1 | Use Copilot to identify and summarize missing values |
| Q2 | Use Copilot to fill missing text columns (Region, Category) |
| Q3 | Fill missing numeric values (Units Sold, Unit Price) |
| Q4 | Create a new calculated column (Total Revenue) |
| Q5 | Use Copilot to find total price of Order ID = ORD0123 |
| Q6 | Use Copilot to generate average profit by region chart |
| Q7 | Create a pivot table by Category and Region |
| Q8 | Build a dashboard with KPIs and charts |
| Q9 | Use Copilot to generate a summary report |

## 📂 Dataset Details
- **Total Records:** 350 Orders
- **Period:** January – December 2023
- **Columns:** Order ID, Region, Country, Category, 
  Sales Channel, Order Priority, Order Date, 
  Units Sold, Unit Price, Unit Cost

## 📊 Key Findings
- **Missing Values Found:**
  - Region: 12 missing → filled with mode
  - Category: 10 missing → filled with "Unknown"
  - Units Sold: 15 missing → filled with median
  - Unit Price: 14 missing → filled with median
- **ORD0123 Total Revenue:** $15,849.30
- **Calculated Column:** Total Revenue = Units Sold × Unit Price
